## Folder Naming Considerations

The folder `WorldlyFineDiscussion` was **NOT renamed** because:
- `main.js` uses relative imports: `require('./search_product.js')`
- Renaming would require updating all imports
- Risk of breaking runtime behavior
- User constraint: "If moving a file risks breaking imports → DO NOT MOVE IT"

If renaming is desired in the future:
1. Update all `require('./filename')` statements
2. Update paths in `search_product.js` for CSV files
3. Test thoroughly before deployment
